# Example of food delivery support chatbot

A primitive chatbot built with Python that helps users resolve common food delivery issues like:

* Missing orders
* Stolen deliveries
* Damaged food

It uses:

* 🧠 Machine Learning (scikit-learn)
* 🔍 Keyword detection
* 💬 Simple conversation memory (including saving username and last question)

---

## 🚀 Features

* Intent detection using NLP
* Keyword fallback system
* Context-aware replies (remembers previous issue)
* Personalized responses (remembers user name)

---

## ⚙️ Installation

1. Clone the repository

2. Install dependencies:

```
pip install -r requirements.txt
```

---

## ▶️ Run the bot

```
python main.py
```
